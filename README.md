# Student Management System

## Requirements
- Java JDK 17 or above
- Download: https://adoptium.net

## How to Run

### Step 1: Clone the repository
git clone https://github.com/abhishekmewada18/student-management-system.git
### Step 2: Go to parent folder
cd student-management-system
### Step 3: Compile
javac StudentManagement/Student.java StudentManagement/StudentManager.java StudentManagement/Main.java

### Step 4: Run
java StudentManagement.Main

## Features
- Add Student
- View Students
- Delete Student
